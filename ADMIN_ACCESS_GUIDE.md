## Common Admin Tasks

### Add a New Product
1. Go to Products section
2. Click "Add Product" button
3. Fill in:
   - Product name
   - Description
   - Category
   - Price (‚Çπ)
   - Stock quantity
   - Image URL
   - Check "Featured" if you want it on homepage
4. Click "Create Product"

### Update Order Status
1. Go to Orders section
2. Click the eye icon next to an order
3. Scroll down to see status buttons
4. Click desired status (Pending ‚Üí Processing ‚Üí Shipped ‚Üí Delivered)
5. Status updates automatically

### Manage Low Stock Products
1. Dashboard shows "Low Stock Items" count
2. Go to Products section
3. Products with stock < 10 are highlighted in red
4. Edit product to update stock quantity

### Deactivate a User
1. Go to Users section
2. Find the user in the table
3. Click the user icon in Actions column
4. User status changes to "Inactive"
5. They won't be able to login until reactivated

## Troubleshooting

**Can't login?**
- Verify you're using: username `admin`, password `admin123`

- Try clearing browser cache and cookies

**Dashboard not loading?**
- Ensure you're logged in

- Refresh the page

- Swipe left/right to scroll tables horizontally
- All data is there, just scroll to see it
